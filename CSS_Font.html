CSS Font Families

CSS에서는 두 가지 글꼴 모음이 있다.

  • generic family - 비슷한 모양의 글씨체 모음 Ex)"Serif" 또는 "Monospace"
  • font family - 특정한 글꼴 Ex)"Time New Roman" 또는 "Arial"
Generic family Font family Description
Serif Times New Roman
Georgia		 Serif fonts have small lines at the ends on some characters
Sans-serif Arial
Verdana		 "Sans" means without - these fonts do not have the lines at the ends of characters
Monospace Courier New
Lucida Console		 All monospace characters have the same width

Note: 컴퓨터의 경우, Sans-serif 글꼴이 serif 글꼴보다 더 읽히기 쉽다.

Font Size

font-size 속성은 텍스트의 크기를 지정한다.

텍스트의 크기를 지정할 수 있는 것은 웹디자인에서 중요하다. 하지만, paragraphs를 headings 처럼 보이게 하거나 headings를 paragraphs 처럼 보이게 하기 위해서 글꼴 크기 조정을 사용하면 안된다.

headings에는 <h1>~<h6>, paragraphs에는 <p>처럼, 항상 적절한 HTML 태그를 사용해라"

font-size 속성은 절대적이거나 상대적일 수 있다.

절대적 크기:

  • 특정한 크기에 텍스트를 지정한다.
  • 사용자가 브라우저 내의 텍스트 크기를 바꿀 수 없도록 한다(접근성에 좋지 않으므로)
  • 절대적 크기는 출력의 실제 크기가 알려졌을 때 유용하다.

상대적 크기:

  • 요소와 상대적인 크기를 지정한다.
  • 사용자가 브라우저 내의 텍스트 크기를 변경할 수 있도록 한다.

Note: 글자의 크기를 지정하지 않는다면 paragraphs일 경우 디폴트 크기는 16px 이다(16px=1em).

Font Variant

font-variant 속성은 텍스트가 작은 대문자로 표기될 것인지 지정한다.

작은 대문자 글꼴에서는 모든 소문자가 대문자로 변환된다. 그러나 변환 된 대문자는 텍스트의 원래 대문자보다 작은 글꼴 크기로 나타난다.
