Text Color

color 속성은 텍스트 색상을 정하는데 사용된다. 색상은 다음과 같이 구체화된다:

  • 색상의 이름 - Ex) "red"
  • HEX 속성-Ex) "#ff0000"
  • RGB 속성-Ex) "rgb(255,0,0)"

페이지의 디폴트 텍스트 색상은 body 셀렉터 안에 정의 한다.

Text Alignment

text-align 속성은 텍스트의 수평적인 정렬을 정할 때 사용된다.

텍스트는 왼쪽, 오른쪽, 중간 또는 justified로 정할 수 있다.

다음의 예시는 중간, 왼쪽 그리고 오른쪽으로 정렬된 텍스트를 보여준다.(텍스트의 방향에 따라 텍스트의 디폴트 정렬이 정해진다)

Example

텍스트 정렬이 "justify"일 때, 각 라인은 각 라인의 너비가 같고 왼쪽과 오른쪽의 여백이 일직선이도록 늘어난다.
