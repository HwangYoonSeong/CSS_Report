CSS 2D Transforms

CSS transforms는 요소를 이동,회전,크기 조절 및 기울일 수 있다.

2D 변형을 보려면 마우스를 올려봐라:

2D
rotate

이번 chapter 에서 다음의 CSS 속성을 배울 것이다.

  • transform

CSS 2D Transforms Methods

transform 속성으로 다음과 같은 2D 변형 method들을 사용할 수 있다:

  • translate()
  • rotate()
  • scale()
  • skew()
  • matrix()

The translate() Method

translate() method는 요소를 현재 위치에서 이동시킨다(주어지는 x축과 y축에 따라)

Example

The rotate() Method

rotate() method는 요소를 주어지는 각도에 따라 시계방향 또는 반시계방향으로 돌린다.

Example

음수를 사용하면 요소를 반시계 방향으로 돌릴 수 있다.

The scale() Method

scale() method는 요소의 크기를 증가 또는 감소시킨다(주어지는 너비와 높이에 따라)

The skew() Method

skew() method는 주어지는 각도를 x축과 y축에 따라 요소를 기울인다.

Example

두번 째 인자가 지정되지 않으면 x축에 따라 기울인다.

The matrix() Method

matrix() method는 모든 2D 변형 method들을 하나로 결합한다.

matrix() method는 수학함수가 포함된 6개의 인자를 사용하여 요소를 회전,크기 조정,이동 및 기울이기를 할 수 있다.

매개변수는 다음과 같다:

matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
