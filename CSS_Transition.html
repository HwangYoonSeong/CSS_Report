How to Use CSS Transitions?

transition 효과를 주기위해, 2가지를 지정해야한다:

  • 변경될 CSS 속성

Specify the Speed Curve of the Transition

transition-timing-function 속성은 transition 효과의 속도 곡선을 지정한다.

transition-timing-function 속성은 다음의 값을 가진다:

  • ease-느리게 시작한 다음 빠르게 다시 천천히 종료하는 transition 효과를 지정(디폴트)
  • linear-시작과 끝이 같은 속도인 transition 효과를 지정
  • ease-in -시작이 느린 transition 효과를 지정
  • ease-out -끝이 느린 transition 효과를 지정
  • ease-in-out -시작과 끝이 느린 transition 효과를 지정
  • cubic-bezier(n,n,n,n) -3차 bezier 함수에서 자신의 값을 정의 할 수 있다.
